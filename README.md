# 南寧白話輸入方案

> * 採用『香港語言學會粵語拼音方案』
> * 聲調輸入 q平 v上 x去 （雙擊爲陽調），邊擦音sl[ɬ]，老派口音的舌尖元音zy[tsɿ]、cy[tsʰɿ]、sy[sɿ]
> * 提供繁簡字體轉換方案，IPA版的方案提示音轉換爲IPA（國際音標）
> * 提供一種反查方案，[`]反查南寧白話
> * PC上按Ctrl+Enter可以將IPA上屏，按Ctrl+Shift+Enter可以將候選第一項的提示音上屏
